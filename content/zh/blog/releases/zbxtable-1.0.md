zbxtable 1.0.1 版本发布
**Bug Fixes**

- 修复 zbxtable 组件与 ms-agent 组件日志权限问题
- 修复 zbxtable 调用 zabbix web，如果 web 服务器为 http 请求 400，图形显示异常问题
- 修复 zbxtable 打包脚本 control 问题
- 移除 zbxtable 配置文件 appurl 配置，便于图形显示

**特性**

- 添加 yum 安装源，可方便安装和更新组件
- 添加 centos6 和 8 支持

安装及更新请查看[使用文档](/docs/)
